Pizza Cost Calculator

Build a webpage where a user can:

  • Select a pizza size (Small, Medium, Large)
  • Select toppings using checkboxes
  • Click a button to calculate the total price
  • See the final total cost displayed on the page

Requirements

1. Pizza Sizes

Create a dropdown list (<select>) with these options:

Pizza Size Price
Small $8
Medium $10
Large $12

2. Toppings

Create at least 5 toppings using checkboxes. Each topping should cost $2.

Example toppings:

  • Cheese
  • Pepperoni
  • Mushrooms
  • Olives
  • Onions

3. Use Arrays (Required)

In your JavaScript file (script.js), you must create an array like this: 

let toppings = ["Cheese", "Pepperoni", "Mushrooms", "Olives", "Onions"];

4. Use a For Loop (Required)

You must use a for loop to check which toppings were selected.

5. Calculate Total Price

Your program should:

  1. Get the pizza size from the dropdown
  2. Find the base price of the pizza
  3. Add $2 for every topping selected
  4. Display the final total

Example:

  • Medium Pizza = $10
  • 3 toppings selected = 3 Ã— $2 = $6
  • Total = $10 + $6 = $16

6. Display Output

When the user clicks the "Calculate Total" button, display:

  • Selected pizza size
  • Selected toppings
  • Total price
