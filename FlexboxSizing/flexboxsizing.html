Create a flex box container layout as per the below rules.

  • The container should have 3 items with 3 colors with text 1,2, 3 in them.
  • The items must be laid out vertically
  • Item A should always take up twice as much of the available vertical space as Item B
  • Item B should shrink, but only half as aggressively as Item C when space is limited
  • Item 3 should always maintain its initial size and should not grow.
