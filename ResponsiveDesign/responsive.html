Create your html page as shown below to demonstrate the responsive design concepts.

  1. The page background should start with background of yellow color.
  2. When the page reaches tablet size (750px), the color should be blue
  3. When the page reaches mobile size (450px), the color should be red

Design a responsive div section.

  1. Create a div section container with 5 items in it (paragraphs or headings)
  2. When the page is bigger( more than 500px), display the 5 items side by side.
  3. When the page is smaller (less than 500 px), display the 5 items vertically

Hint : Use Media queries as shown below.

@media (max-width: 600px) {

body { background-color: lightsalmon; }

}
