Using alert and prompt commands, create a pizza order.

  1. Put an alert saying "Welcome to Pizza Shop!". Select your own pizza shop name.
  2. Use an alert to display the prices of Small, Medium, Large Pizzas
  3. Use a prompt to ask the user for the size of pizza they want (Small, Medium, Large) and store them in variables
  4. Let the user order 3 pizzas
  5. Calculate the total cost based on the size of pizzas ordered
